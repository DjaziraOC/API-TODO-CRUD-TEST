# Todo Frontend (React)

Frontend SPA pour l’API backend (JWT + CRUD tasks).

## Prérequis
- Node.js 18+

## Démarrage

```bash
cd frontend
npm install
npm run dev
```

- Frontend : <URL frontend>
- Backend attendu : <URL backend>

## Configuration API
Créer (optionnel) :
- `frontend/.env` avec

```env
VITE_API_URL=<URL backend>
```

Si absent, le frontend utilise <URL backend> par défaut.
